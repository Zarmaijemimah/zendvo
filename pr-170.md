## Summary
Added a new currency conversion utility at `src/lib/currencyConverter.ts` to fetch the current NGN/USDC market rate, cache it for 10 minutes, and expose helper functions for converting NGN amounts to USDC.

This feature improves the UX for mixed-fiat and crypto flows by providing a reliable on-demand multiplier that avoids unnecessary external API calls.

Closes #170

## What Changed
- Created `src/lib/currencyConverter.ts`
  - Fetches USDC/NGN price from a configurable external endpoint (`CURRENCY_CONVERTER_API_URL`), defaulting to CoinGecko.
  - Caches the computed NGN→USDC rate in memory for 10 minutes.
  - Provides `getNgnToUsdcRate`, `convertNgnToUsdc`, and `clearCurrencyConverterCache`.
- Added unit tests in `__tests__/lib/currencyConverter.test.ts`
  - Validates API fetch behavior
  - Validates in-memory caching and TTL expiration
  - Validates conversion math and invalid input handling

## How Has This Been Tested?

- Ran Jest against the new utility:
  - `node node_modules/jest/bin/jest.js __tests__/lib/currencyConverter.test.ts --runInBand`
- Confirmed the new utility returns a valid number and reuses cache within the TTL.

## Notes
- `CURRENCY_CONVERTER_API_URL` may be used to override the default CoinGecko endpoint in environments where a private or paid exchanger is preferred.
